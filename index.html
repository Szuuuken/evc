About

Auf dieser Seite könnt ihr die multiple choice Fragen aus der LVA Einführung in Visual Computing durchgehen und lernen.

Die Fragen können als Liste betrachtet werden oder mit dem "Trainer" geübt werden.

Ich habe evc bereits abgeschlossen und damit liegt nun die Weiterentwicklung und pflege der Fragen bei euch. Mehr Informationen zum Mitmachen findet ihr im Abschnitt mitmachen.

Die Seite ist auf github gehostet.

Fragenkatalog

Die Fragen kommen aus einer viel zahl an Quellen wie dem vowi, informatik-forum, Studenten und so weiter.

Des weiteren werden die Fragen in die Kategorien Test1 und Test2 eingeteilt.

Ich kann leider nicht garantieren, dass alle Fragen aus allen Tests vorhanden und korrekt sind. Jedoch sind 95% aller ins System eingetragenen Fragen korrekt. Falls ihr falsche Fragen findet oder neue Fragen zum System hinzufügen wollt lest euch den Abschnitt mitmachen durch.

Mitmachen

Es gibt eine viel zahl an Möglichkeiten wie du helfen kannst die Seite verbessern kannst, es muss nicht nur programmiert werden.

Pflege der Fragen

Es gibt mehrere Möglichkeiten wie du dich um die Pflege der Fragen kümmern kannst:

  • ein issue (bug) für eine falsche Frage erstellen
  • einen issue (feature request) für eine neue Frage erstellen
  • eine Frage im code korrigieren (js/fragen.js)
  • eine neue Frage im code hinzufügen (js/fragen.js)

Fragen von der LVA-Leitung prüfen lassen

Bei einigen Fragen kommt es immer wieder zu längeren Diskussion, ob sie nun korrekt ist oder nicht. Deswegen wurde die Idee geboren, sich die Liste aller Fragen auszudrucken und in die Sprechstunde der LVA-Leitung zu gehen um die Korrektheit zu überprüfen.
